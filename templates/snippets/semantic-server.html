The Semantic server is an open-source REST API wrapper that seamlessly interfaces to RDF entities stored as subject graphs in Fedora 4, expanded and enriched search through Elastic Search, supported by a full SPARQL endpoint using Fuseki

Fedora Commons

The core of the semantic server is Fedora Commons Repository used a Linked Data Platform. Linked Data is stored and managed as subject RDF graphs in Fedora 4, with the most work done with using BIBFRAME and Schema.org vocabularies.
